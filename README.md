# cp-sat-lp-trace-viewer

Offline viewer for CP-SAT LP trace visualization.

GitHub Pages serves the site from `docs/`.

The deployed app includes:

- a timeline viewer for structured CP-SAT traces
- bundled LP-focused examples
- JSON upload for traces generated on another machine

Once Pages is enabled, the site URL is:

- `https://takehidesoh.github.io/cp-sat-lp-trace-viewer/`
